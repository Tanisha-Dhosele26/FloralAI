# 🌸 FloralAI - Smart Flower Recommendation System

## 📌 Overview
FloralAI is a full-stack web application designed to help users choose the perfect flowers based on emotions, occasions, and relationships. The platform enhances user experience by providing personalized bouquet recommendations, add-on gift selections, and AI-generated messages.

This project aims to simplify the process of selecting flowers by combining intuitive UI design with intelligent recommendation logic.

## 🚀 Features

### 🌼 Flower Recommendation
- Get bouquet suggestions based on occasion, mood, and relationship
- Dynamic UI updates based on user input

### 🎁 Add-On Gifts
- Select additional items like:
  - Chocolate 🍫
  - Teddy 🧸
  - Greeting Cards 💌

### 💬 AI Message Generator
- Generates personalized messages based on selected flowers and gifts
- (Gemini API integration planned)

### 📖 Flower Encyclopedia
- Explore flowers with meanings and descriptions
- Interactive cards with modern UI

### 🛒 Order Summary
- Displays selected flowers, add-ons, and message
- Includes pricing system

### 🔐 Authentication (UI Ready)
- Login & Signup pages designed
- Backend integration in progress
